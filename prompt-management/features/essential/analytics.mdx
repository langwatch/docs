## Overview Metrics

Track key usage statistics:

- **Traces**: Total number of prompt executions
- **Threads**: Number of conversation threads
- **Users**: Number of unique users

## LLM Metrics

Monitor your AI model usage:

- **LLM Calls**: Number of API calls made
- **Total Cost**: Cost of all API calls
- **Tokens**: Total tokens consumed

## Sentiment Analysis

View the sentiment distribution of your prompt inputs with a visual breakdown showing positive, neutral, and negative sentiment percentages.

## Activity Timeline

Track your prompt usage over time with a simple line chart showing trace activity patterns.
