LangWatch provides an audit log to track user actions and changes in the platform. This allows you to monitor and audit the activities of your users and ensure compliance with your organization's policies.

## Overview

The audit log is a record of all actions performed in the platform. It is stored in a database and can be queried to find specific actions and changes.

You can search the audit log by:
- **Name**: Filter by user name or identifier
- **Action**: Filter by specific action types
- **Date**: Filter by date range to find actions within a specific time period

Additionally, you can download the audit log data as a CSV file for further analysis or compliance reporting.
